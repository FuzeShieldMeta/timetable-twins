# Timetable Twins - Availability Calendar Generator

A tool for merging multiple calendar files to find free time slots between multiple people and generating a unified calendar with availability events.

## Features

- **Multi-Calendar Upload**: Upload multiple `.ics` calendar files simultaneously
- **Timezone Conversion**: Automatically converts UTC times to Auckland timezone (UTC+12/+13) with DST support
- **Free Time Extraction**: Identifies when each person is free between their scheduled events
- **ICS File Export**: Generates a merged `.ics` file with all availability events
